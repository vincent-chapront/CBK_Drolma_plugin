# CBK Drolma Plugin

This plugin adds features to be used on a wordpress site with elementor.

# Features

## Courses management

This feature adds an administration menu to configure courses, as well as widgets to display them.

A course is defined by its title, a description and a list of lesson. A lesson is defined by a date and an optional name.

There are 2 elementor-compatible widgets.

- CBK Drolma Course Extract: for a given course, show a configurable number of past and future lesson.
- CBK Drolma Course Calendar: for a given course, show the complete list of lesson for a configurable date range
